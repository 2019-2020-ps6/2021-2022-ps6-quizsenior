Tutoriel
  • Bienvenue dans ce tutoriel pour vous apprendre à jouer. Pour passer à l'étape suivante, une fois les informations assimilées, appuyez sur la touche "Entrée", pour réécouter la consigne, appuyez sur la barre "Espace" et enfin, pour revenir en arrière, appuyez sur la barre "Effacer".


    • Bon tutoriel !
  • Choisissez votre thème, il représente une catégorie de question (exemple: SPORT)


  • Ensuite, choisissez votre questionnaire, il correspond au sujet des questions (exemple: Football)


  • Comment naviguer pendant le quiz ?
    Il vous suffit de vous déplacer avec les touches directionnelles de votre clavier
    •
  • Comment lire la question ou lire les réponses ?
    Il vous suffit de soit vous déplacer, soit appuyer sur la barre espace de votre clavier
    •
  • Comment passer aux réponses, une fois avoir bien compris la question, et comment valider vos réponses ?
    Il vous suffit d'appuyer sur la touche "Entrée" de votre clavier.
    •
  • Félicitations, vous êtes fin prêts à vous lancer dans vos premiers quiz.
